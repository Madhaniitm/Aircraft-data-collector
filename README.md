# Aircraft Data Collector

A React + Node.js prototype for collecting aircraft information from Wikipedia.

Features:
- Add aircraft names manually or upload a list
- Choose detail fields to fetch
- Search Wikipedia and gather infobox data
- View uploaded aircraft and display fetched parameters

## Run locally
1. Open the workspace in VS Code
2. Run `npm install`
3. Run `npm run dev`
4. Open the client URL printed by Vite
